# 00 Audit

This folder is the external audit entrypoint for RED-Python.

## Instruction

Use [02_AUDITORIA_EXTERIOR_CONTRACT_FIRST.md](./02_AUDITORIA_EXTERIOR_CONTRACT_FIRST.md) as the single source of truth for any external audit.
Do not declare an external audit complete unless the contract phases are satisfied and the repo state matches the reported evidence.

## Scope

- Repo profile and legacy control inventory
- Contract claims and evidence
- Operational trace and human proof
- Claim/evidence matrix
- Mapping to active rules and guards
- Adversarial findings
- Verdict and remediation plan
